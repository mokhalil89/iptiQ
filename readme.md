# coding challenge assignment for Quality Engineer

##Objective

Create a test automation framework using either Java + Rest Assured + Cucumber or POSTMAN from scratch for the following requests from https://reqres.in/ 
Run test through command line
Run the tests nightly (every 24 hr.) or on each commit.
Use library of your choice to display test execution report
Please share the solution prefereably in a Github repository. 



## Installation

Use the maven command line to install framework.

```bash
mvn clean install
```

## Running Testcases

Use the maven command line to run testcases

```bash
mvn test
```
